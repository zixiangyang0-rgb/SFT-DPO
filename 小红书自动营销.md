## 项目经历

### 小红书智能社交媒体营销系统

**所属公司：** [公司名称]（工程机械配件外贸出口）

**项目时间：** 2025年01月 — 至今

**项目简介：**
针对公司海外工程机械配件出口业务，独立从零设计并搭建了一套基于AI与浏览器自动化技术的全链路小红书（Xiaohongshu）智能营销系统，覆盖内容创作、分发、互动、数据分析全流程，实现海外品牌曝光与询盘转化的自动化闭环。

---

**一、技术架构设计与核心功能实现**

1. **Chrome DevTools Protocol 自动化引擎**
   - 基于 WebSocket 协议连接 Chrome 浏览器，通过 CDP 命令直接操控网页 DOM，实现内容自动发布、评论互动、用户信息抓取等操作，无需第三方平台 API，稳定性与成功率更高。
   - 自主封装 `XiaohongshuPublisher` 类，统一管理 CDP 连接生命周期、会话状态、错误重试，具备超时控制、自动重连、断线告警等生产级健壮性。

2. **多账号隔离与安全管理**
   - 实现 Chrome Profile 目录级账号隔离，各小红书账号 Cookie 独立存储、互不干扰。
   - 支持通过命令行一键切换账号，账号数量可横向扩展，满足业务团队多账号矩阵运营需求。
   - 登录状态自动缓存（TTL 12小时），到期前自动重新校验，大幅减少重复登录开销。

3. **内容发布全流程自动化**
   - 支持图文与视频两种发布模式，图片/视频可来自 URL 自动下载（自动添加 Referer 绕过防盗链）或本地文件直传。
   - 自动识别正文中话题标签（`#标签` 格式），逐个写入话题输入框并等待平台建议加载，确保标签命中准确。
   - 支持定时发布，自动化填写创作者中心定时控件，自动翻页翻日历交互。
   - 2026年2-3月小红书创作者中心改版期间，快速定位并更新了发布按钮、定时开关、日期输入框、多图上传、正文编辑器等5类核心选择器，保障业务不中断，体现了对平台变动的高度敏感性与快速响应能力。

4. **防检测人机模拟**
   - 全链路引入随机时序抖动（Jitter），每次操作延迟在基础值 ±25% 范围内随机化，避免固定间隔被平台识别为机器人行为。
   - 话题标签逐字符输入、每字随机停顿，模拟真实用户打字节奏。
   - 支持无头（Headless）与有窗口两种运行模式，未登录时自动从无头切换为有窗口触发扫码登录。

5. **内容数据分析与商业洞察**
   - 接入小红书创作者中心数据分析接口，自动化抓取每篇笔记的曝光量、观看次数、封面点击率、点赞、评论、收藏、涨粉、分享、人均观看时长等9项核心指标。
   - 数据自动导出为 CSV 文件，供业务团队进行周报/月报统计与趋势分析，替代人工逐篇手动抄录，效率提升显著。

6. **内容检索与竞品分析**
   - 实现关键词搜索、首页推荐流抓取、笔记详情读取（含滚动加载评论）功能，可按最新/最多点赞/最多收藏等维度筛选。
   - 支持用户主页快照与笔记列表抓取，便于运营团队快速了解竞品账号的内容策略与互动数据。

7. **评论互动自动化**
   - 支持对目标笔记发表一级评论、按条件（评论ID/作者/文本片段）精准回复指定评论。
   - 支持对笔记执行点赞/取消点赞、收藏/取消收藏等互动操作，帮助账号维持活跃度与热度。

8. **跨平台与部署灵活性**
   - 支持 Windows/macOS/Linux 三平台运行，Chrome 路径自动探测（支持 Google Chrome 与 Chromium）。
   - 支持远程 CDP 连接（`--host` / `--port` 参数），可将 Chrome 部署在远程服务器，脚本在本地通过 IP:Port 远程控制，适配多种企业 IT 环境。
   - 支持 WSL + Windows UNC 路径混用场景，解决跨系统文件上传路径兼容问题。

9. **进程安全与并发控制**
   - 引入单实例锁机制，防止同一账号同时运行多次发布流程导致数据冲突或异常。
   - 发布完成后自动清理临时下载文件与缓存，防止磁盘空间泄漏。

**二、业务对接与价值产出**

- 将 AI 自动化能力直接对接海外营销业务：产品上新时，运营人员提供产品图片与英文卖点文案，系统自动完成多语言适配、本地化内容生成（结合 AI 辅助）、图片处理、平台发布全流程，发布效率提升约5倍。
- 内容数据分析模块与公司周报体系对接，每周一自动导出上周内容表现数据，运营团队据此调整选题与发布策略，形成"数据驱动内容优化"闭环。
- 多账号矩阵运营：建立3个以上定位差异化的账号（产品展示号/行业知识号/客户案例号），各账号内容风格与受众不同，系统通过账号切换机制统一管控，降低人力运营成本。
